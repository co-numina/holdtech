## Detailed Description
HoldTech analyzes Solana token holder bases in seconds — right from your browser.

🔬 WHAT IT DOES
• Scans the top holders of any Solana token
• Detects wallet bundles (same funding source, same slot buys)
• Measures concentration (top 5/10/20 holder %)
• Flags fresh wallets (<7 days old)
• Generates a letter grade (A–F) and quality score (0–100)
• AI-powered verdict explaining what the data means

🧩 TWO WAYS TO USE
1. POPUP — Click the icon on any page, paste a contract address, get an instant score
2. AUTO-INJECT — Visit a token on pump.fun, DexScreener, Birdeye, or Solscan and the score badge appears automatically in the bottom-right corner

📡 SUPPORTED SITES
• pump.fun
• DexScreener (Solana pairs)
• Birdeye
• Solscan
• Defined.fi

⚡ FEATURES
• Works with single-page apps (SPA-aware navigation detection)
• Results cached for 5 minutes to minimize API calls
• Click "Full analysis →" to see detailed charts, deep scan, and AI verdict on holdtech.xyz
• Light and dark mode aware

🛡️ PRIVACY
• No data collected. No tracking. No accounts required.
• Only calls the HoldTech API when you scan a token or visit a supported page.
• Open source: github.com/co-numina/holdtech
